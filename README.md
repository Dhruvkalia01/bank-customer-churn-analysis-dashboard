# Bank Customer Churn Analysis Dashboard

## Overview
This project is an interactive Power BI dashboard developed using SQL and MySQL to analyze customer churn trends, customer segmentation, risk distribution, and income-based behavior.

## Project Objectives
- Analyze customer attrition trends
- Identify high-risk customer segments
- Perform income-category analysis
- Create business-focused KPI dashboards
- Generate actionable business insights

## Dashboard Features
- Total Customers KPI
- Attrition Rate KPI
- Average Credit Limit KPI
- High Risk Customers KPI
- Customer Segmentation by Card Type
- Income Category Analysis
- Risk-Level Distribution

## Tools & Technologies
- Power BI
- SQL
- MySQL
- DAX
- Data Visualization

## Key Insights
- Attrition Rate: 16.07%
- High Risk Customers: 4.61%
- Blue card holders showed highest customer count
- Lower income categories showed higher attrition concentration
