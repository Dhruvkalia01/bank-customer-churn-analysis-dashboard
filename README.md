# Bank Customer Churn Analysis Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed using SQL and MySQL to analyze customer churn behavior, customer segmentation, risk distribution, and income-based trends in banking data.

The dashboard helps identify customer attrition patterns and provides actionable business insights for decision-making.

## Business Problem
Banks face customer attrition due to multiple factors such as customer risk profile, income category, and card usage behavior. This project analyzes customer data to identify churn trends and high-risk customer groups.

## Objectives
- Analyze customer attrition trends
- Identify high-risk customer segments
- Perform customer segmentation analysis
- Analyze churn distribution by income category
- Create business-focused KPIs and dashboards
- Generate actionable business insights

## Dashboard KPIs
- Total Customers
- Attrition Rate %
- Average Credit Limit
- High-Risk Customers %

## Dashboard Features
### Customer Segmentation by Card Type
Analyzes customer distribution across Blue, Silver, Gold, and Platinum card categories.

### Customer Attrition Distribution
Shows the proportion of existing vs attrited customers.

### Customer Risk Distribution
Visualizes low, medium, and high-risk customer segments.

### Attrition by Income Category
Analyzes churn behavior across different income groups.

## Tools & Technologies
- Power BI
- SQL
- MySQL
- DAX
- Data Visualization

## Key Insights
- Overall attrition rate observed: **16.07%**
- High-risk customers account for **4.61%**
- Blue card customers form the largest customer segment
- Lower income categories show relatively higher attrition concentration
